# 🌱 AI Eco Product Scanner

## Project Idea
AI Eco Product Scanner is a web application that analyzes the environmental impact of everyday products.

Users enter a product name and the AI provides:

- Material used
- Recyclability
- Environmental impact
- Eco-friendly suggestions
- Eco score

## Technologies Used
- Python
- Streamlit
- OpenAI API
- GitHub

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py

## Example

Input:
Plastic Water Bottle

Output:
Material: Plastic  
Recyclability: Yes  
Environmental Impact: High plastic waste  
Suggestion: Use reusable bottles  
Eco Score: Medium
